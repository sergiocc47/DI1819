Gestión Restaurante

Esta aplicación sirve para gestionar un restaurante.

Esta página es la ayuda principal. La aplicación consta de doce ventanas:

  • Pantalla Principal
  • Apertura Ticket
  • Gestión Mesas
  • Alta Mesa
  • Gestión Carta
  • Alta Producto Carta
  • Gestión Productos Ticket
  • Alta Producto Ticket
  • Filtrado Ticket por Fecha
  • Filtrado Ticket por Mesa
  • Visualización Tickets por Fecha
  • Visualización Tickets por Mesa
Cada una de las ventanas tiene su propia página de ayuda.
